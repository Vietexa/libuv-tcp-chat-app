# TCP Chat App example, made with libuv

This is a TCP chat app example made with libuv as a way for me to learn how to use libuv.

I've decided to open source this project as a learning opportunity for the people who need
to work with TCP and libuv.

I left a lot of comments in the TCP chat client example so you can easily understand what's going on. I still haven't covered the whole TCP chat server example, in the way I did with the TCP chat client, but you can take that as a challenge.
I hope you learned something from my example, the code is released under a permissive MIT license.

## How to build
### Arch Linux

``mkdir build``

``cd build``

``cmake ..``

``make``
